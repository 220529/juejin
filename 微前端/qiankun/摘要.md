# Qiankun 微前端源码解析系列 - 文章摘要

## 00-Qiankun项目介绍
Qiankun 是蚂蚁金服开源的微前端框架，基于 single-spa 封装，提供 HTML Entry、JS 沙箱、样式隔离等能力。本文介绍 3.0 版本的项目结构和核心 API。

## 01-Qiankun架构总览
深入分析 loadApp 核心流程，理解子应用从加载到挂载的完整链路，对比 registerMicroApps 和 loadMicroApp 两种加载模式。

## 02-沙箱机制
分析 @qiankunjs/sandbox 包的实现，包括 StandardSandbox、Membrane 代理模式、Compartment 代码执行隔离，以及事件监听、定时器等副作用补丁。

## 03-应用加载
深入 @qiankunjs/loader 的流式加载实现，分析 TagTransformStream、WritableDOMStream、脚本沙箱包装和 defer 脚本处理机制。

## 04-生命周期
分析子应用生命周期管理，包括 beforeLoad、bootstrap、mount、unmount 等钩子的执行顺序，以及生命周期函数的多种获取方式。

## 05-通信机制
介绍主子应用通信方式，包括 Props 传递、全局状态管理、自定义事件和共享依赖，提供最佳实践建议。

## 06-路由劫持
分析基于 single-spa 的路由劫持机制，包括 History API 劫持、activeRule 匹配、应用状态流转和 History 监听补丁。

## 07-预加载策略
深入预加载机制，分析 prefetch 配置、requestIdleCallback 空闲加载、分级预加载策略和自定义预加载实现。
