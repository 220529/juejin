# 无界微前端源码解析系列 - 文章摘要

## 00-Wujie项目介绍
无界是腾讯开源的微前端框架，基于 Web Components + iframe 实现原生隔离，本文介绍项目结构和核心概念。

## 01-Wujie架构总览
深入 startApp 启动流程，分析 WuJie 沙箱类的创建、激活和启动机制，理解双容器分离架构。

## 02-沙箱机制
分析 iframe 沙箱的创建和 patch 机制，包括 history 劫持、事件分流、脚本 IIFE 包装等核心技术。

## 03-CSS隔离
深入 Shadow DOM 实现 CSS 隔离的原理，包括 Web Component 定义、:root 转换、@font-face 提取等。

## 04-JS隔离
分析 Proxy 代理 window、document、location 的实现，理解 DOM 操作如何从 iframe 代理到 Shadow DOM。

## 05-路由同步
深入主子应用路由同步机制，分析 URL Query 存储、短路径优化、History 劫持和前进后退处理。

## 06-通信机制
分析 EventBus 事件总线和 props 数据传递的实现，理解跨应用通信和嵌套场景的事件共享。

## 07-插件系统
深入 loader 和 hook 机制，分析 CSS/JS 处理、资源排除、事件钩子和属性覆盖等扩展能力。
