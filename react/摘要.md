# React 源码系列文章摘要

## 00-环境搭建
开箱即用的 React 源码调试项目，支持 v18/v19 双版本。clone 后执行 pnpm dev 即可启动，通过 webpack alias 映射源码，断点调试无障碍。

## 01-源码全景
React 源码系列总览篇。介绍 Scheduler、Reconciler、Renderer 三大模块，Fiber 结构、Lane 优先级、双缓冲机制，建立源码全局视角。

## 02-useState
深入 useState 源码，解析 Hook 链表结构、mountState 首次渲染、dispatchSetState 触发更新的完整流程，揭秘 Eager State 优化机制。

## 03-useEffect
深入 useEffect 源码，解析 Effect 环形链表、flags 标记、依赖比较机制。对比 useLayoutEffect 执行时机，解答闭包陷阱等常见问题。

## 04-Fiber工作循环
解析 React 渲染核心流程。workLoopSync 与 workLoopConcurrent 的区别，beginWork 递阶段向下遍历，completeWork 归阶段创建 DOM。

## 05-Diff算法
解析 React Diff 的实现策略。单节点先比 key 再比 type；多节点三轮遍历处理更新、增删、移动；通过 lastPlacedIndex 判断节点移动。

## 06-Scheduler
揭秘 React 可中断渲染的实现。5 级优先级与超时时间、taskQueue 双队列、最小堆结构、5ms 时间切片、MessageChannel 调度触发。

## 07-Commit
解析 Commit 阶段三个子阶段：Before Mutation 读取 DOM 状态，Mutation 执行 DOM 增删改，Layout 执行 useLayoutEffect 和生命周期。

## 08-自动错误边界插件
实现一个 Babel 插件，自动为 React 组件注入 ErrorBoundary HOC。通过 AST 转换，遍历 JSX 元素，过滤原生标签，创建 HOC 包装。
