# Lerna 源码解析系列 - 文章摘要

## 00-Lerna项目介绍
Lerna 是一个快速、现代的 monorepo 管理工具，用于管理和发布多个 JavaScript/TypeScript 包。本文介绍 9.x 版本的项目结构、核心命令和配置方式。

## 01-Lerna架构总览
深入分析 Command 基类和命令执行流程，理解 Project、Package、ProjectGraph 等核心抽象，以及拓扑排序执行机制。

## 02-版本管理
分析 lerna version 命令的实现，包括版本号解析（交互式、语义化提交、命令行指定）、依赖更新、Changelog 生成和 Git 操作。

## 03-发布流程
深入 lerna publish 命令，分析 from-git、from-package、canary 等发布模式，以及依赖链接解析、打包、发布和 2FA 处理。

## 04-变更检测
分析基于 Git 的变更检测机制，包括获取上次 tag、diff 变更文件、映射到包、依赖传递和过滤选项（scope、ignore、ignore-changes）。

## 05-任务编排
深入 lerna run 和 lerna exec 的实现，分析拓扑排序算法、并发控制、循环依赖检测和流式输出机制。

## 06-Nx集成
分析 Lerna 与 Nx 的集成，包括项目图构建、任务缓存原理、远程缓存配置和分布式执行，以及性能优化最佳实践。
